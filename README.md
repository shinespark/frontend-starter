# frontend-starter

## How to install

```sh
$ npm install
```

## Check npm scripts

```sh
$ npm run
```

## gulp

### gulp default

```sh
$ npm start
```

### gulp *

```sh
$ npm run-script gulp *
```

### other added scripts

```sh
$ npm run-script *
```

See "scripts" in the package.json.
And, see [https://docs.npmjs.com/misc/scripts]
